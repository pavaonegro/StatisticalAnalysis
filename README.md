# StatisticalAnalysis
Python notebooks for statistical analysis (normality &amp; hypothesis tests) and VR trajectory visualization used in the user experiment

This repository contains all analysis code used in the publication “Enhancing Realism in Virtual Paleontological Experiences: The Role of Spatially Accurate Impulse Responses”.
It includes statistical tests, normality checks, HMD trajectory analysis, and plotting routines implemented in Python/Jupyter.
